# 📊 연령별 인구 현황 대시보드

행정안전부 주민등록 연령별 인구현황 데이터를 기반으로, 원하는 지역의 인구 구조를 살펴보고 인구구조가 비슷한 지역을 찾아주는 Streamlit 웹앱입니다.

## 🖥️ 스크린샷

> (배포 후 스크린샷을 이 자리에 추가해보세요)

## ✨ 주요 기능

### 1. 지역별 연령 인구 그래프
- 사이드바에서 지역을 **검색(직접 입력) + 선택(드롭다운)** 방식으로 고를 수 있어요.
- 전체 / 남성 / 여성 인구를 선택해서 볼 수 있고, 남녀를 겹쳐서 비교하는 것도 가능해요.
- Plotly 기반 인터랙티브 꺾은선 그래프로, 마우스로 확대·축소하고 값을 바로 확인할 수 있어요.

### 2. 인구구조가 비슷한 지역 Top 5
- 각 지역의 **연령대별 인구 비율(구성비)**을 계산해서, 지역 크기와 상관없이 나이대 분포의 '모양'만 비교해요.
- 선택한 지역과 다른 모든 지역 사이의 **유클리드 거리**를 계산해 구조가 가장 비슷한 5개 지역을 찾아줘요.
- 선택 지역(강조 표시) + Top5 지역의 연령 구성비를 겹쳐 그린 라인차트로 한눈에 비교할 수 있어요.
- Top5 지역의 구조 차이 크기를 막대그래프로도 보여줘요.


## 🛠️ 사용한 데이터

- **출처**: 행정안전부 주민등록 연령별 인구현황 (월간)
- **단위**: 읍·면·동 (시도·시군구 합계 행도 함께 포함되어 있음)
- **주요 컬럼**:
  - `행정구역`: 지역명 + 행정구역 코드
  - `연령별_계_0세` ~ `연령별_계_100세 이상`: 전체 인구
  - `연령별_남_0세` ~ `연령별_남_100세 이상`: 남성 인구
  - `연령별_여_0세` ~ `연령별_여_100세 이상`: 여성 인구

> ⚠️ 데이터에는 시도·시군구 합계 행도 섞여 있어서, Top5 유사 지역 결과에 읍면동이 아닌 상위 행정구역이 함께 나올 수 있습니다.

## 🚀 로컬에서 실행하기

```bash
# 1. 필요한 라이브러리 설치
pip install -r requirements.txt

# 2. 앱 실행
streamlit run main.py
```

## ☁️ Streamlit Cloud 배포 방법

1. 이 저장소를 GitHub에 올립니다. (`main.py`, `requirements.txt`, CSV 파일이 같은 폴더에 있어야 해요)
2. [share.streamlit.io](https://share.streamlit.io) 에 접속해 GitHub 계정으로 로그인합니다.
3. "New app"을 클릭하고 이 저장소를 선택합니다.
4. Main file path에 `main.py`를 지정하고 배포(Deploy)합니다.
